# Steps to Upload the Project

### Step 1: Create a GitHub Account
If you do not already have one, create a GitHub account at **https://github.com**.

### Step 2: Create a New Repository
- Click **New Repository**.
- Enter the repository name.
- Add a short description.
- Choose **Public** visibility.
- Initialize the repository with a README if needed.

### Step 3: Upload Project Files
Upload all Markdown files, documentation, screenshots, and images into their respective folders.

### Step 4: Commit Changes
Write a meaningful commit message, such as:

```
Initial commit - AI Content Marketing using UGC Ads project
```

Click **Commit Changes**.

### Step 5: Verify Repository
Check that:
- All files are uploaded.
- The folder structure is correct.
- The README displays properly.
- Images and documentation open correctly.
