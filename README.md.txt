# AI Content Marketing using UGC Ads
## Prompt Engineering Task 2 (2026) – Future Interns

## Project Overview
This project demonstrates how Artificial Intelligence (AI) and Prompt Engineering can be used to create authentic, engaging, and high-converting User Generated Content (UGC) advertisements for real businesses. Unlike traditional advertisements that often feel overly promotional, UGC ads are designed to resemble genuine customer experiences, making them more relatable and trustworthy. The project focuses on building a reusable AI prompt system that generates marketing content suitable for social media platforms such as Instagram Reels, Facebook Ads, and YouTube Shorts.

For this project, the selected product is **Mamaearth Ubtan Face Wash**, a popular skincare product known for its natural ingredients like Turmeric and Saffron, which help remove tan and brighten the skin. Using structured prompts, AI generates multiple advertisement hooks, scripts, captions, and call-to-actions that can be directly used by brands, content creators, and marketing agencies.

## Project Objective
The main objective of this project is to develop a reusable prompt engineering system capable of generating professional-quality UGC advertisements with minimal manual effort. The system is designed to:
- Generate multiple attention-grabbing hooks.
- Create short UGC-style advertisement scripts.
- Follow the Problem → Solution → Call-to-Action (CTA) framework.
- Produce platform-specific marketing content.
- Generate captions and effective CTAs for social media campaigns.
- Provide reusable prompts that can be adapted for different brands and products.

## Project Workflow
The project follows a structured workflow:
1. Select a real product or business.
2. Define the target audience and marketing goal.
3. Create AI prompts using prompt engineering techniques.
4. Generate multiple hooks and UGC ad scripts.
5. Create captions, hashtags, and CTAs.
6. Optimize content for Instagram Reels, Facebook Ads, and YouTube Shorts.
7. Review and refine the generated content for better engagement.
